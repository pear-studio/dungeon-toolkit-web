# 开发规范

本目录收录所有**开发过程中需要遵守的规范**，与 `docs/` 根目录下的项目文档（部署说明、架构文档）分开管理。

## 规范列表

| 文件 | 内容 |
|------|------|
| [`data-sources.md`](./data-sources.md) | 规则数据来源规范——所有 D&D 数据必须从 `rules_source/` 读取 |
| [`frontend.md`](./frontend.md) | 前端开发规范——禁止硬编码游戏数据、组件命名、状态管理 |

## 文档 vs 规范

| | `docs/` 根目录 | `docs/conventions/` |
|---|---|---|
| **定位** | 项目文档（给部署者/使用者看） | 开发规范（给开发者看） |
| **内容** | 部署步骤、架构说明、API 文档 | 编码约定、数据流规范、禁止行为 |
| **示例** | `deployment.md` | `data-sources.md`, `frontend.md` |
