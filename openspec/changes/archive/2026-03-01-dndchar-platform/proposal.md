
# Dungeon Toolkit · 项目提案

## 一句话描述

一个面向中文 D&D 5e 玩家的开源工具箱，支持向导式角色创建与角色卡管理。

---

## 背景与动机

- 现有工具（DndBeyond、Roll20）均为英文，订阅制，不开源
- 中文 D&D 玩家缺乏一个开源、中文友好的角色管理工具
- 项目同时作为开源项目发布，需要对小白用户友好的一键部署体验

---

## 目标用户

- 中文 D&D 5e 玩家（活跃用户预期几千人，并发量不高）
- 希望 fork 自部署的技术用户（需要详细文档 + Docker Compose 一键部署）

---

## 核心功能范围

### Phase 1：MVP

- [ ] 用户系统（邮箱+密码注册登录，JWT 认证）
- [ ] 向导式角色创建（8步，见下方详细设计）
- [ ] 角色卡查看与编辑
- [ ] Docker Compose 一键部署 + 详细部署文档

### Phase 2：核心体验完善

- [ ] 角色公开分享（唯一链接，只读）
- [ ] 移动端适配优化

### Phase 3：锦上添花

- [ ] PDF 导出标准角色卡
- [ ] Excel 等格式导出
- [ ] 角色广场（浏览公开角色）
- [ ] D&D 5e 2024版规则数据支持

---

## 非目标（明确不做）

- 虚拟桌游/战斗模拟功能
- 实时多人协作
- 多语言支持（仅中文）
- 移动端 App（响应式网页即可）
- D&D 5e 以外的规则系统（框架预留，暂不实现）

---

## 技术栈决策

| 层级 | 技术选型 | 决策原因 |
|------|---------|---------|
| 前端 | React + TypeScript + Tailwind CSS | 功能全面，AI 辅助开发友好，移动端响应式 |
| 后端 | Django + Django REST Framework | 开发者熟悉 Python，生态成熟 |
| 数据库 | PostgreSQL | JSONB 字段适合半结构化角色数据，全文搜索支持 |
| 缓存 | Redis（可选） | 会话管理、API 响应缓存 |
| Web 服务器 | Nginx | 反向代理 + 静态资源服务 |
| 容器化 | Docker Compose | 小白友好，一键部署 |
| CI/CD | GitHub Actions | 自动测试 + 部署到腾讯云 |
| AI 接入 | 可配置（OpenAI / Claude / 通义千问） | 通过环境变量切换，用户自备 API Key |

---

## 部署目标

- 生产环境：腾讯云 CVM
- 开源用户：Docker Compose 一键启动（`git clone` → `cp .env.example .env` → `docker-compose up -d`）
- 文档：详细的中文部署文档，面向零基础用户

---

## 竞争差异化

1. **开源 + 自部署**：数据完全自主，无订阅费
2. **小白友好**：一键部署，详细文档
3. **中文优先**：面向中文玩家设计，无语言障碍
