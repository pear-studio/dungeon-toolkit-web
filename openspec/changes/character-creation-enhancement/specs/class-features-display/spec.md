## ADDED Requirements

### Requirement: 按等级展示职业特性
职业步骤中，当玩家选中一个职业并设定了等级后，系统 SHALL 展示该职业从第 1 级到当前等级所有获得的职业特性名称与简短描述，按等级升序排列。

#### Scenario: 展示单职业特性列表
- **WHEN** 玩家选择「法师」职业，等级设为 3
- **THEN** 展示第 1 级「法术施放」「奥术恢复」，第 2 级「奥术传统」的特性条目

#### Scenario: 多职业特性合并展示
- **WHEN** 玩家配置「法师 3 级 + 战士 2 级」
- **THEN** 法师特性和战士特性分组展示，各自标注职业来源

#### Scenario: 数据缺失时的降级处理
- **WHEN** 某职业特性描述字段为空
- **THEN** 仍展示特性名称，描述区域显示「—」占位，不崩溃
